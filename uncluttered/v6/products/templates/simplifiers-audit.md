### Migration Checklist Template

Use this for each migration:

#### Migration: [Old Tool] ‚Üí [New Tool]

**Pre-Migration**
- [ ] Export all data from [Old Tool]
- [ ] Verify export completeness (spot check 5-10 records)
- [ ] Document current workflows that depend on [Old Tool]
- [ ] Identify integrations that connect to [Old Tool]
- [ ] Set up [New Tool] account and configure basics
- [ ] Map data fields: Old ‚Üí New

**Data Transfer**
- [ ] Import data to [New Tool]
- [ ] Verify data integrity (counts match, content correct)
- [ ] Test core workflows in [New Tool]
- [ ] Set up integrations in [New Tool]
- [ ] Configure permissions / team access

**Switchover**
- [ ] Run both tools in parallel for [X days]
- [ ] Train team on [New Tool] if needed
- [ ] Update all bookmarks, shortcuts, documentation
- [ ] Update any Zapier/automation workflows
- [ ] Notify team of switchover date

**Post-Migration**
- [ ] Confirm everything works for [X days]
- [ ] Export final backup from [Old Tool]
- [ ] Cancel [Old Tool] subscription
- [ ] Update tool inventory
- [ ] Document any lessons learned
