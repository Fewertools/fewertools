The 4-step audit

A framework for cutting without regret.

1

List everything

Export your subscriptions. Check your browser bookmarks. Search your email for receipts. You'll be surprised what you find.

  • Check credit card statements
  • Review browser extensions
  • List mobile apps you pay for
2

Score each tool

For each tool, ask: When did I last use this? What would break if it disappeared? Is there overlap with something else?

  • Daily use = essential
  • Weekly use = evaluate
  • Monthly or less = cut candidate
3

Find consolidation opportunities

Which tools overlap? One good tool often replaces two mediocre ones. Look for "swiss army knife" tools that do multiple jobs well.

  • Notion can replace 5+ single-purpose tools
  • Linear can replace Jira + Trello + Asana
  • Zapier can eliminate manual busywork
4

Cut ruthlessly

Cancel the subscriptions. Delete the apps. You can always re-subscribe if you actually need it. Spoiler: you usually won't.

  • Set a "tool budget" and stick to it
  • One tool per job, maximum
  • If in doubt, cut it out
