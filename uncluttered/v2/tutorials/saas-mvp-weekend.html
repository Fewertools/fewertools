1. Project Setup

Let's start by creating a new Next.js project with all the tools we need.

Create the project

Open your terminal and run:

npx create-next-app@latest my-saas --typescript --tailwind --app
cd my-saas

Select the default options when prompted. This gives us:

  • Next.js 14 with App Router
  • TypeScript for type safety
  • Tailwind CSS for styling

Install dependencies

npm install @supabase/supabase-js @supabase/auth-helpers-nextjs stripe @stripe/stripe-js

Set up environment variables

Create a .env.local file:

# Supabase
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# Stripe
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
STRIPE_SECRET_KEY=your_stripe_secret_key
STRIPE_WEBHOOK_SECRET=your_webhook_secret

2. Authentication with Supabase

üü¢
Supabase

Open source Firebase alternative with PostgreSQL database and built-in auth.

Get Started Free ‚Üí

Create a Supabase project

  1. Go to supabase.com and create a new project
  2. Wait for the database to be provisioned (~2 minutes)
  3. Go to Settings ‚Üí API and copy your URL and anon key

Create the Supabase client

Create lib/supabase.ts:

import { createClient } from '@supabase/supabase-js'

export const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL!,
  process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!
)

Build the auth UI

Create app/login/page.tsx:

'use client'
import { useState } from 'react'
import { supabase } from '@/lib/supabase'

export default function Login() {
  const [email, setEmail] = useState('')
  const [loading, setLoading] = useState(false)

  const handleLogin = async (e: React.FormEvent) => {
    e.preventDefault()
    setLoading(true)
    
    const { error } = await supabase.auth.signInWithOtp({ email })
    
    if (error) {
      alert(error.message)
    } else {
      alert('Check your email for the login link!')
    }
    setLoading(false)
  }

  return (
    <form onSubmit={handleLogin}>
      <input
        type="email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        placeholder="you@example.com"
        required
      />
      <button type="submit" disabled={loading}>
        {loading ? 'Sending...' : 'Send Magic Link'}
      </button>
    </form>
  )
}

3. Database & API

Supabase gives us a full PostgreSQL database. Let's create a table for our SaaS data.

Create the database schema

In Supabase SQL editor, run:

-- Create profiles table
create table profiles (
  id uuid references auth.users primary key,
  email text,
  plan text default 'free',
  created_at timestamp with time zone default now()
);

-- Enable Row Level Security
alter table profiles enable row level security;

-- Users can only see their own profile
create policy "Users can view own profile"
  on profiles for select
  using (auth.uid() = id);

-- Trigger to create profile on signup
create function handle_new_user()
returns trigger as $$
begin
  insert into profiles (id, email)
  values (new.id, new.email);
  return new;
end;
$$ language plpgsql security definer;

create trigger on_auth_user_created
  after insert on auth.users
  for each row execute procedure handle_new_user();

4. Stripe Payments

üí≥
Stripe

Payment infrastructure for the internet. Handle subscriptions, invoices, and more.

Get Started Free ‚Üí

Create a Stripe product

  1. Go to Stripe Dashboard
  2. Create a new Product with a recurring price
  3. Copy the Price ID (starts with price_)

Create checkout API route

Create app/api/checkout/route.ts:

import { NextResponse } from 'next/server'
import Stripe from 'stripe'

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)

export async function POST(req: Request) {
  const { priceId, userId } = await req.json()

  const session = await stripe.checkout.sessions.create({
    mode: 'subscription',
    payment_method_types: ['card'],
    line_items: [{ price: priceId, quantity: 1 }],
    success_url: `${process.env.NEXT_PUBLIC_URL}/dashboard?success=true`,
    cancel_url: `${process.env.NEXT_PUBLIC_URL}/pricing`,
    metadata: { userId }
  })

  return NextResponse.json({ url: session.url })
}
