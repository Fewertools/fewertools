Overview

Supabase gives you everything you need to build a backend in minutes. It's built on Postgres—the world's most trusted database—and adds authentication, file storage, edge functions, and realtime subscriptions on top.

Unlike Firebase, you're not locked in. Your data lives in a real Postgres database that you can query with SQL, export anytime, and even self-host if you want. It's the best of both worlds: Firebase-like developer experience with Postgres power.

Key Features

Postgres Database

Full Postgres with Row Level Security.

Authentication

Email, social logins, magic links, MFA.

Storage

S3-compatible object storage with CDN.

Realtime

Subscribe to database changes instantly.

Edge Functions

Serverless TypeScript on Deno Deploy.

Row Level Security

Fine-grained access control at database level.
