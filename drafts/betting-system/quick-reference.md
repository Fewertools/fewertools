# Betting Quick Reference Card
*Pin this. Check it before every bet.*

---

## Value Formula
```
Value exists when: YOUR probability > IMPLIED probability + 5%
```

## Implied Probability from Odds
```
Decimal:  1/odds × 100
American+: 100/(odds+100) × 100  
American-: odds/(odds-100) × 100
```

| Odds | Implied % |
|------|-----------|
| 1.50 | 67% |
| 1.80 | 56% |
| 2.00 | 50% |
| 2.20 | 45% |
| 2.50 | 40% |
| 3.00 | 33% |
| 4.00 | 25% |

---

## Quarter Kelly Stake
```
Stake % = [(odds-1) × prob - (1-prob)] / (odds-1) ÷ 4
```

**Shortcut Table (Quarter Kelly):**

| Your Edge | Approx Stake |
|-----------|--------------|
| 5% | 1% bankroll |
| 10% | 2% bankroll |
| 15% | 3% bankroll |
| 20% | 4% bankroll |

## Hard Limits
- Max single bet: **5%**
- Max daily exposure: **15%**
- Daily stop loss: **-10%**

---

## Before Betting Checklist
- [ ] Analysis done (not just gut)
- [ ] My probability estimated FIRST
- [ ] Edge ≥ 5%
- [ ] Stake follows system
- [ ] Within daily limits
- [ ] Not emotional/tilted

---

## Red Flags = NO BET
- 3+ key players missing
- 3rd game in 7 days
- Derby match
- Dead rubber
- Post-international break

---

## After a Loss
1. Did I follow the system?
2. If yes → variance, move on
3. If no → why? fix it
4. Never chase

---

*Trust the process. Play the long game.*
