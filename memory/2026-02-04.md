**Key technical details:**
- Uses The Odds API (free: 500 req/month) for live bookmaker odds across 15+ bookmakers
- Uses football-data.org (free: 10 calls/min) for team statistics
- Poisson model calculates true scoreline probabilities
- Form adjustment (last 5 matches) with configurable weight
- Value threshold: 5%+ edge required (accounts for model uncertainty)
- Quarter Kelly staking for optimal bankroll growth

**Test results:** Model passes sanity checks:
